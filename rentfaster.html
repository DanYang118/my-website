RentFaster - UX Evaluation & Redesign

Background

RentFaster is a rental platform where users reported difficulties with search, filtering, and application flows.

Goal

Improve the end-to-end rental experience by identifying pain points in the user journey and providing practical design solutions.

Solution

Mapped user journeys for different personas. Proposed UX improvements: simplify filtering options, display listing dates, improve photo accuracy, streamline registration, and add reminders for landlord response.
