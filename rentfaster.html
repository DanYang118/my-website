RentFaster

Background

RentFaster is a rental platform where users reported difficulties with search, filtering, and application flows.

The Goal

Improve the end-to-end rental experience by identifying pain points in the user journey and providing practical design solutions.

Solution

Mapped user journeys for different personas. Proposed UX improvements: simplify filtering options, display listing dates, improve photo accuracy, streamline registration, and add reminders for landlord response.

Core Competencies Applied

UX Evaluation
Conducting Heuristic Evaluations and Competitor Auditing to pinpoint usability gaps in the existing rental search flow.
Problem Identification
Translating messy user feedback into actionable design requirements, focusing on filtering logic and information density.
User Empathy Mapping
Developing detailed Personas and Journey Maps to visualize the emotional highs and lows of the "Calgary Rental Struggle."
Audience Analysis
Analyzing the specific needs of newcomers and students in Calgary to optimize the "First-time Renter" application experience.
