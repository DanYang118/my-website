Project Management App - Dashboard Design

Try this prototype

Background

Team project to design a desktop project management tool supporting task organization, scheduling, and collaboration. Collaborated with Hana(Task part)

Goal

Provide users with a clear overview of projects and tasks through an interactive dashboard.

My Role

Focused on designing the Dashboard, including: Project overview with key metrics and progress charts. Project status visualization. Interactive components for switching projects and tracking deadlines.

Design Process

  • Research: Analyzed competitor tools (Trello, Monday, Asana) to identify gaps and best practices.
  • User Flow Mapping: Defined the journey from adding tasks → assigning → tracking progress.
  • Wireframes: Created low-fidelity layouts to explore information hierarchy and layout options.
  • Iterations: Tested wireframes with peers and refined navigation and visual hierarchy.
  • Design System: Built reusable components (cards, charts, buttons, widgets) with a consistent color palette, typography, and grid system to unify the app.
  • High-Fidelity Mockups: Developed polished screens in Figma, applying the design system for consistency.
  • Prototype: Built an interactive prototype in Figma to demonstrate key features such as project switching and project status visualization.
  • Usability Testing (Plan): Conducted peer testing sessions on the interactive prototype, gathering feedback on clarity, task flow, and navigation; refined layouts and interactions to improve efficiency and user satisfaction.
